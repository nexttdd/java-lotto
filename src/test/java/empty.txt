#기능 요구사항
1. 로또발급
    - 로또 발급 장수 구하기
        : 입력받은 금액으로 로또발급 장수 구하기
        : 발급가능여부 CHECK ( 최소금액 확인 )

    - 로또 발급하기
        : 1 ~45 사이에서 6개 숫자 랜덤으로 발급하기 >> DONE
            - 로또번호들 : 1 ~ 45 숫자 보관 >> DONE
            - 랜덤생성기 : 1 ~ 45 숫자 섞은뒤 6개 숫자 반환 (?) >> DONE

        - 로또 객체
            : 발급한 6개 숫자 저장 (로또번호 list)
                - 로또번호
                    : 1~45사이.  아니면 에러 발생
        - 로또들 저장 객체
            : 1급 콜랙션, 로또 list 저장

        : 로또들 객체를 반환

2. 당첨번호 저장
3. 당첨개수 확인
4. 수익률 계산
