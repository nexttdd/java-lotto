#기능 요구사항
1. 로또발급
    - 로또 발급 장수 구하기 (로또머니 객체) >> DONE
        : 입력받은 금액으로 로또발급 장수 구하기
        : 발급가능한 최소금액 판단

    - 로또 발급하기 >> DONE
        : 1 ~45 사이에서 6개 숫자 랜덤으로 발급하기 >> DONE
            - 로또번호들 : 1 ~ 45 숫자 보관 >> DONE
                *셔플이랑, 정렬을 LottoBalls객체에 구현 추가
            - 랜덤생성기 : 로또티켓 한장 발급 / 여러장 발급

        - 로또 객체 >> DONE
            : 발급한 6개 숫자 저장 (로또번호 list)
                - 로또번호 >> DONE
                    : 1~45사이.  아니면 에러 발생
                    : 중복검사 - 숫자 정렬할 때 검사하도록 함.

        - 로또들 저장 객체 >> DONE
            : 1급 콜랙션, 로또 list 저장

        : 로또들 객체를 반환

2. 당첨번호 저장 >> DONE
    - 로또티켓 객체를 이용해서 지난주 당첨번호 저장
    - LottoTicketMachine 에 수동으로 로또티켓 생성하는 기능 추가

3. 당첨개수 확인
    - 당첨개수 체크하는 서비스 필요
    - 당첨개수별로 저장하는 객체

4. 수익률 계산
    - 수익률 계산하는 서비스 필요
        - 실제 수익률 계산하는 거는  당첨개수 저장 객체에서 해야겠다.
